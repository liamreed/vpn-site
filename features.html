Private

  • Custom personal VPN server setup to your needs.
  • Server in place of your choice setup to delete logs automatically.
  • 10tb/month bandwidth (100mb/s port) after which speed is limited to 1mb/s. (Not shared so the 10tb applies only to you).
  • Connect via OpenVPN.

Shared

  • Join a vpn network with other users. (Shared IP)
  • Server setup to delete logs automatically.
  • Shared server 10tb/month bandwidth (100mb/s port) after which speed is limited to 1mb/s. (Bandwidth abusers will be removed).
  • Connect via OpenVPN.

Multiple encrypted servers across the globe

We offer a range of server locations via our VPN, with servers in France, Canada and the USA being available immediately. You can request a server location via our order form and we will try to source a suitable server optimal for price and performance.

Multiple protocols

Get your VPN setup how you want by a professional. We offer OpenVPN, PPTP and L2TP protocols and all servers send all logs straight to /dev/null/ by default. Or for added anonymity, join a shared VPN with shared IP's.
