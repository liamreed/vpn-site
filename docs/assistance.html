Requesting Assistance

Speed issues

If you find download speeds to be slow on the Private unshared plan then please contact support@anovpn.com and we will switch you over to an optimised configuration. If you find speeds to be slow on the shared plan then please contact support and we will investigate bandwidth usage on your server node.

Please note that p2p traffic on shared plans is not guaranteed but allowed if it is not excessive and a hindrance to other users on the node. We will contact you if you usage is excessive.

Other issues/Connection issues

If you have another issue or connection issue, then please contact support@anovpn.com with your problem as the subject. We will get back to you as soon as we can and investigate network issues if one arrises.
